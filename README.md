## Installation & Run Locally


Clone the project

```bash
  git clone git@github.com:hosamhamdy258/CLI_DBMS.git
```

Go to the Project directory

```bash
  cd my-project
```

Start the project run in terminal 

```bash
  ./dbms_v1.0.sh
```
- note : if your stuck in any menu just press ctrl-d to go back
